- 한국형 에꼴 42 코디세이는 스스로 학습과 동료 평가를 통해 코딩 능력과 문제해결 역량을 향상하는 교육방식입니다. 
- **코디세이 프로그래밍** 강좌에서는 Codyssey LMS(Learning Management System)를 활용하여 학생 스스로 학습하고, 과제를 해결한 후 동료 평가를 통해 검증받는 방식으로 진행됩니다
